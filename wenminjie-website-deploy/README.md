# Wen Minjie Portfolio Website

Personal portfolio site for Wen Minjie (文敏婕), PhD researcher at QUT.

## Live site
https://wenminjie.space

## Tech stack
- Pure HTML/CSS (single file)
- No build step required
- Deployed on Vercel
- Custom domain: wenminjie.space

## Update content
Edit `index.html`, commit, and Vercel auto-deploys within 1 minute.

## Add real photos
1. Create folder `images/`
2. Upload photos: `hero.jpg`, `work-01.jpg` ... `work-05.jpg`
3. In `index.html`, find the `<img src="...">` lines and change the URL to `images/hero.jpg` etc.
